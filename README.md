🛡️ Cybersecurity Awareness Training Simulator

A self-hosted educational platform designed to teach users how to recognize phishing attempts and improve digital security awareness through safe, simulated exercises.

📌 Overview

This project provides interactive cybersecurity training scenarios where users are exposed to simulated social engineering patterns in a controlled environment.

The goal is to educate users on:

Recognizing suspicious login pages
Identifying fake email or message patterns
Understanding common phishing techniques
Building safe browsing habits

⚠️ This project is strictly for educational purposes and does not interact with or impersonate real online services.

🎯 Features
🧪 Interactive training modules
📚 Cybersecurity awareness lessons
🧠 Scenario-based learning exercises
📊 Progress tracking for trainees
🔐 Safe simulated login interfaces (no real credentials involved)
🌐 Fully self-hosted deployment
🚀 How It Works

Users are presented with mock security scenarios that replicate common social engineering patterns in a safe sandbox environment.

Each session includes:

A simulated login interface (clearly labeled as training)
Educational prompts explaining red flags
Immediate feedback on user choices
Security tips and best practices
🛠️ Tech Stack
Frontend: HTML / CSS / JavaScript (or React optional)
Backend: Node.js / Express (optional)
Database: SQLite / MongoDB (for progress tracking)
Hosting: Docker-ready deployment
